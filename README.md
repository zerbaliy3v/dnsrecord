## Installation & Usage

```
git clone https://github.com/zerbaliy3v/dnsrecord.git
cd dnsrecord
pip install -r requirements.txt
chmod +x dnsrecord.py
sudo cp dnsrecord.py /usr/bin/dnsrecord
dnsrecord -d examole.com
```
